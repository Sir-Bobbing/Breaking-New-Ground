# Cassette Secrets
There are some cassettes which can only be obtained by renaming one in an anvil.

The following is a list of the presently existing *secret* Cassettes.
- **Zam** - Produces the Tenna Dance Gif.
- **color_test** - Creates a glitchier version of the Cassetteless Television.
- **Skeleton** - "RAAAH! *BANG BANG BANG*."
- **Leftchaotix** - gives you a breakdancing Steve.
- **Little_Pianist** - "Chicken Jockey!" -Steve
