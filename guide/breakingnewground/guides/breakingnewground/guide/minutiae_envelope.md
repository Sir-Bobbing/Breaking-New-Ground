# Envelope

### Pigeon

Friendly creature, that can be found in mountain areas or near a village

- Likes seeds of various kind, and will happily follow someone if they have some
- Hunted by Wild Cats, Ocelots and Foxes

### Pigeonhole

A cozy house for pigeons to rest inside of, and leave their, ehm... “byproducts”

### Letter

Good alternative to a book, if you don’t have much to say  
Just keep it away from foxes...

### Package

Slightly inconvenient (but pretty cheap) way to reduce inventory clutter  
Convenient for deliveries, though

### Sealing

A way to ensure an author and integrity of a Letter or Package

### Mailbox

Entrypoint into the fascinating world of mail delivery

- Every Mailbox in the overworld must have an address
- Connecting a Mailbox to the mail network costs 3 experience levels
- Stores received mail
- Allows sending mail

### Address Tag

- Use the item to open Address Selection GUI
    - All available addresses are shown
- Use on a mailable item in GUI to set the recipient
- Can be used on a Mailbox to change its address

### Delivery

When mail and seeds are inserted into a Mailbox, its front hatch will open, allowing nearby Pigeons to pick mail up and start a delivery

- Pigeons can deliver mail (and return to) unloaded chunks without issues
- Delivering Pigeons ascend high above the clouds to make traveling long distances more efficient, thus they can only be seen while nearby a Mailbox

### Payback Tag

- Used on a mailable item to set the requested payback
- Anything that fits in the Package can be requested

Allows for a safe trade of items:

- Transaction will be handled by Mail Service
- Buyer receives Payback Box for the payment
- Mail will be kept in storage untill the buyer pays for it
- If buyer does not pay in time - mail will be returned back to sender

### Services

Virtual addresses that can receive mail and perform various actions

- Includes mail crafting system and recipes
- Extendable by datapacks, addons or KubeJS scripts
